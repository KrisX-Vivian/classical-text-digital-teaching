# 《红楼梦》题库JSON格式优化报告 (v2.2)

## ✅ 本次优化解决的问题

### 问题1: 选择题选项重复
**优化前**: 选项既在`options`数组中，又杂糅在`stem`里

**优化后**: 选项完全从`stem`中移除，只在`options`数组中

```json
// 优化后
{
  "content": {
    "stem": "下列文学常识的表述，不正确的一项是（ ）",
    "options": [
      { "label": "A", "text": "选项A内容..." },
      { "label": "B", "text": "选项B内容..." },
      { "label": "C", "text": "选项C内容..." },
      { "label": "D", "text": "选项D内容..." }
    ]
  }
}
```

---

### 问题2: 材料与问题杂糅
**优化前**: 材料和问题全部放在`stem`中

**优化后**: 材料放在`material`字段，问题放在`stem`字段

```json
// 优化后
{
  "content": {
    "material": "①材料一...\n②材料二...\n③材料三...",
    "stem": "作者用刘姥姥的视角描写了大观园中的三处居所。请依次写出居所名称，并根据居所环境分析其主人的性格特质。"
  }
}
```

---

### 问题3: 答案和解析杂糅
**优化前**: 答案和解析混在`stem`中

**优化后**: 自动提取到`answer`字段

```json
// 优化后
{
  "answer": {
    "correct": "①第一处：潇湘馆...",
    "analysis": "①第一处这段描写出自《红楼梦》第四十回...",
    "keyPoints": []
  }
}
```

## 🚀 前端渲染示例

### 分析题（有材料）

```vue
<template>
  <div class="analysis-question">
    <!-- 材料部分 -->
    <div v-if="question.content.material" class="material-section">
      <h4>阅读材料</h4>
      <div class="material-content">{{ question.content.material }}</div>
    </div>

    <!-- 问题部分 -->
    <div class="question-section">
      <p class="stem">{{ question.content.stem }}</p>
    </div>

    <!-- 答案部分 -->
    <div v-if="showAnswer" class="answer-section">
      <div v-if="question.answer.correct">
        <p><strong>答案:</strong> {{ question.answer.correct }}</p>
        <p v-if="question.answer.analysis">
          <strong>解析:</strong> {{ question.answer.analysis }}
        </p>
      </div>
      <div v-else class="no-answer">
        <p>答案待补充</p>
        <button @click="generateWithAI">AI生成答案</button>
      </div>
    </div>
  </div>
</template>
```
