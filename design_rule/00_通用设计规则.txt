基于现有网页文件结构（index/resource/courses/exercises.html+assets 素材文件夹），在已有 Bootstrap 新中式基础上，升级打造兼具「国家级科研项目专业性」「古典文本人文肌理」与「新生代数字设计美学」的专属网页，核心约束：不改动现有文件命名、功能模块、页面框架及代码注释，所有新增样式存入 assets/css/，新增素材（图片 / 图标 / 动效）按规范存入 assets 对应子文件夹，最终输出可直接部署的完整静态版本，打开 index.html 即可查看效果，无需额外配置，适配 PC 端、移动端、平板端多终端访问。
一、风格内核与视觉基底
风格主线：以「科研理性 + 文人文感性」为双核心，锚定「极简高级新中式 + 古典文本数字化特色」，拒绝网红化、低质化视觉元素，摒弃繁复国风堆砌，做到 “古风不复古，简约不简单”。
视觉绑定：色彩体系、视觉符号、交互逻辑深度绑定「古典文本数字化」核心，融合古籍肌理、水墨意境与数字科技感，形成项目专属视觉语言体系。
二、全局色彩系统（精准色值，国风低饱和高级调）
主色：赭石棕（#8B4513，RGB (139,69,19)）+ 米白宣纸色（#FAF0E6，RGB (250,240,230)），主色占比 20%-30%；赭石棕用于标题、线条、交互高亮，米白为全网页基础背景色。
辅助色：朱砂红（#DC143C，RGB (220,20,60)）+ 墨灰（#363636，RGB (54,54,54)），朱砂红用于按钮、重点标注、导航高亮，墨灰为正文主色，全网页零散色彩不超过 5 种。
中性色：浅灰（#F5F7FA）、淡墨浅灰（#E8E5E1），用于模块分割、卡片背景、阴影过渡；文字与背景对比度符合 WCAG 无障碍标准（最小 4.5:1）。
交互色彩规则：按钮 hover 时赭石棕加深 10%（#753810），点击时叠加 10% 淡墨浅灰；导航项选中 /hover 时用朱砂红高亮；全网页色彩体系统一，仅通过辅助色占比微调形成页面视觉节奏。
三、全局版式与布局规范（栅格化 + 国风留白，沿用 Bootstrap）
网格系统：沿用 Bootstrap 响应式 12 列网格布局，PC 端栅格间距 24px，移动端 16px，所有功能模块贴合栅格对齐；仅核心视觉模块（Hero 区、核心成果区）可适度偏移 1-2 列形成视觉焦点。
留白策略：核心内容区留白占比≥30%，首页 Hero 区留白≥50%；文字段落行间距为字号的 1.6-1.8 倍，字间距 0.6-1px，贴合古籍排版的阅读呼吸感。
字体层级规范（PC 端 / 移动端按比例缩放）：
一级标题（项目 / 页面标题）：32-36px/24-28px，思源宋体 / 仿宋，字重 700，赭石棕；
二级标题（模块标题）：20-24px/16-18px，思源宋体，字重 600，赭石棕；
正文（模块内容）：15-16px/14px，思源黑体 / 仿宋，字重 400，墨灰；
辅助文字（说明 / 备注）：12-13px/12px，字重 300，淡墨灰。
四、全局技术适配要求
响应式适配：适配 1920px、1366px、768px、375px/414px 主流分辨率，无内容溢出、布局错乱；移动端自动隐藏非核心国风装饰，优先展示文字与核心功能。
性能优化：首屏加载 PC≤2 秒、移动端≤3 秒，通过图片懒加载、代码压缩实现；核心动效采用 CSS3 实现，禁用多余 JS 插件；兼容 Chrome、Firefox、Safari、Edge 主流浏览器。
无障碍适配：支持键盘 Tab 键导航，所有图片添加 alt 属性，教学视频添加字幕，色彩对比符合无障碍标准。
【01_index_首页优化.txt】（约 1300 字）
基于【00_通用设计规则.txt】要求，优化 index.html 首页（项目介绍页），仅修改样式与动效，不改动现有内容模块顺序、功能及代码注释，新增 CSS 样式存入 assets/css/，所有视觉元素贴合「古典文本数字化」特色，突出项目核心定位与国风高级感。
一、视觉设计精细化要求
Hero 区（首屏核心区）：
背景：全屏 16:9 水墨山水长卷（《千里江山图》风格），叠加从下到上的赭石棕（#8B4513）渐变遮罩（透明度 0→0.6），保证文字清晰性；
文字：主标题 “典籍新诠・古典文本数字化教学平台” 用 36px 思源宋体，赭石棕，下方加 1px 细金线分隔；副标题 “国家级大学生创新创业训练计划项目 —— 以 AI 赋能古典文学教学，用科技重构文化传承方式” 用 16px 思源黑体，墨灰，行高 1.8；
动效：页面加载时，水墨背景从模糊到清晰渐变，文字从下往上水墨淡入（时长 0.8s）。
核心创新点模块：
布局：3 列栅格布局，贴合 Bootstrap 规范，列间距 24px（PC 端）/16px（移动端），留白占比≥35%；
卡片：书卷造型卡片，米白背景 + 赭石棕 1px 细边框，卡片内文字按字体层级规范排布，核心关键词（如 AI 智能解析、沉浸式教学）用「加粗 + 朱砂红 + 轻微宣纸背景高亮」突出；
动效：滚动到可视区域时，卡片从下往上水墨淡入 + 轻微上移（时长 0.5s），每个卡片延迟 0.1s 依次出现，hover 时轻微上浮（y 轴偏移 5px）+ 墨灰阴影加深（模糊半径 8px，不透明度 10%）。
团队 / 成果展示模块：
团队展示：国风角色卡片，竹简造型，包含「圆形头像（赭石棕细描边）+ 姓名 + 角色 + 核心贡献」，核心贡献用朱砂红标注，摒弃单纯罗列学历形式；
成果展示：国风卡片式 + 可视化组合，卡片含「成果名称 + 核心数据 + 示意图 + 查看详情按钮」，按钮为国风圆角样式（圆角 8px），卡片 hover 效果与创新点卡片一致。
二、交互体验精细化要求
导航交互：PC 端顶部导航栏为「固定 + 水墨透明渐变」，初始透明（露出背景），滚动时渐变为米白宣纸色不透明；导航项 hover 时触发「朱砂红下划线渐变动画」（从左到右填充，时长 0.3s），保留现有导航项数量与顺序。
按钮交互：所有功能按钮（查看详情、了解更多）采用国风圆角矩形（圆角 8px），默认状态「赭石棕低透明度填充 + 无边框」，hover 时「赭石棕 100% 填充 + 1.02 倍缩放 + 朱砂红 1px 描边」，点击时「0.98 倍轻微下沉 + 墨灰阴影加深」。
滚动交互：页面采用平滑滚动（scroll-behavior: smooth）；长页面设置「书卷造型回到顶部按钮」，滚动超过 500px 时赭石棕淡入，点击后平滑返回顶部。
三、差异化记忆点设计
在首页次屏设计「典籍新诠价值罗盘」（国风罗盘造型，赭石棕边框、米白盘面），罗盘可旋转，每个扇区对应学术价值、教学价值、社会价值，点击扇区展示详细内容，融合古典罗盘与数字交互，成为首页标志性视觉元素，不占用核心展示区。
四、内容呈现要求
语言风格：兼顾科研严谨性与古典文本人文性，核心创新点、研究价值用「一句话总结 + 分点拆解」呈现，分点符号用「中式圆点 / 竹简纹点」替代常规符号；
文字总量：PC 端一屏文字≤500 字，移动端一屏≤300 字，现有大段文案做古籍式分行拆分，杜绝堆砌；
关键词强化：每个模块提炼 1-2 个核心关键词（如文本数字化、跨学科融合），格式统一，形成记忆强化。
