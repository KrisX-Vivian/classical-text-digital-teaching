    第三步：关键说明（必看）
1. 代码使用方法
按前面的文件结构创建文件夹和文件，把对应代码复制进去；
所有标注<!-- 需手动替换：XXX -->的地方，都是你需要补充的内容（图片、视频、文案、AI 接口等）；
直接用浏览器打开index.html就能看到效果，无需额外配置（引入的是在线 Bootstrap，联网即可）。
2. 需手动补充的核心素材清单
类型	存放路径	需补充的内容
图片	assets/images/	首页背景图、书籍封面、团队照片、视频封面、红楼梦人物关系网等
视频	assets/videos/	教学视频样本（MP4 格式）或视频平台嵌入链接（B 站 / 腾讯视频）
文案	各页面代码中	项目简介、人物介绍、习题答案解析、小程序调研结果、团队信息等
功能对接	exercises.html（AI 模块）	AI 接口对接（学习通 AI / 自研 AI）、题库答案自动判分（可选）
